# 🚦 Smart Traffic Management System using YOLOv8

## 📌 Project Description
This project is an AI-based Smart Traffic Management System that uses YOLOv8 for real-time vehicle detection and traffic analysis. The system processes video input to identify and classify vehicles such as cars, bikes, buses, and trucks.

Based on the detected vehicles, it calculates traffic density and dynamically adjusts traffic signal timing. This helps reduce congestion, minimize waiting time, and improve overall traffic efficiency.

The system also includes advanced components like vehicle tracking, lane-based management, and emergency vehicle prioritization, making it closer to real-world intelligent traffic systems.

## ⚙️ Features
- Real-time vehicle detection using YOLOv8  
- Vehicle classification (car, bike, bus, truck)  
- Traffic density analysis  
- Adaptive signal timing system  
- Emergency vehicle prioritization  
- Annotated output video generation
